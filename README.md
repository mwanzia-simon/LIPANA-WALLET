**Simonyx Wallet** is a sleek and secure digital wallet built for seamless payments and smart money management. It features real-time transaction handling through **Lipana.dev API**, delivering a fast and intuitive fintech experience.

### 🛠 Tech Stack
- **Frontend:** HTML, CSS, JavaScript  
- **Backend:** Node.js with Express  
- **Database:** MongoDB  
- **Payments Integration:** Lipana.dev API
